# Installation

## Installation instructions for users
For direct installation with all dependencies, run on the terminal or command line
```sh
git clone https://github.com/LSDOlab/CADDEE_alpha.git
```

```sh
cd caddee_alpha
```

```sh
pip install -e .
```
